# Traefik

Traefik is a modern HTTP reverse proxy and load balancer that makes deploying microservices easy. Traefik integrates with your existing infrastructure components (Docker, Swarm mode, Kubernetes, Marathon, Consul, Etcd, Rancher, Amazon ECS, ...) and configures itself automatically and dynamically.

## Features

- **No dependency hell**: Single binary made with go, all dependencies are included.
- **Hot-reloading**: No need to restart the process to reload the configuration.
- **WebUI**: Monitor the status of your traefik instance with an elegant dashboard.
- **Let's Encrypt**: Automatically request and use certificates from Let's Encrypt.
- **Backends**: Supports multiple backends like Docker, Swarm mode, Kubernetes, Marathon, Consul, Etcd, Rancher, Amazon ECS, ...
- **Circuit breakers**: Gracefully handle over-capacity.
- **Websocket**: Support for websocket.
- **HTTP/2**: Support for HTTP/2.
- **GRPC**: Support for GRPC.
